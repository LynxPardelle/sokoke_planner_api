## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

There is no need to install, but you can do it anyways, I'm just a sign I can't stop you.

```bash
# install
$ npm install
```

## Running the app

To run the application, you can use Docker for both development and production environments. This is the recommended approach:

## With Makefile (Recommended)

- For Development:

  ```bash
  make dev
  ```

- For production:

  ``` bash
  make prod
  ```

### With Docker Compose

- For development:

  ```bash
  docker-compose up sokoke_planner_api-dev --watch
  ```

- For production:

  ```bash
  docker-compose up sokoke_planner_api-production
  ```

### Without Docker (Old Way)

If you prefer not to use Docker, you can run the application manually:

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
